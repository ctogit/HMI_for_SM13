Descripción detallada

Esta función recibe la llamada desde el software HMI principal cada vez que necesita enviar u obtener datos de la unidad remota RTU. El ciclo de intercambio de tramas se realiza a demanda del cliente, en este caso el software HMI mediante el presente módulo.

Parámetros
a_HMIDataByteVariable tipo array que recibe los datos tipo -int, float- para enviar a RTU.
a_HMIDataStringVariable tipo array que recibe los datos tipo -String- para enviar a RTU.
b_connectVariable tipo boleana que indica el estado de la conexión con la RTU 0: Desconexión 1: Conexión
s_sockVariable tipo <socket> que almacena la identidad del socket generado en la conexión establecida con la RTU.
s_ipVariable tipo string que recibe la dirección s_ip que se ingresó para la RTU.
s_portVariable tipo string que recibe el puerto que se ingresó para la RTU.
Devuelve
a_RTUData Variable tipo array que retorna los datos recibidos desde la RTU.
